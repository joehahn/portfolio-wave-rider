How to read the winner: this sweep runs over the published post-COVID window (2022-03-31 → 2025-10-31), which is dominated by a single position (RKLB) that ran in 2025, so total return on this window keeps rising as λ falls toward 0, because a smaller variance penalty lets the optimizer concentrate harder into the single 2025 winner (RKLB). That is the in-sample tail, not proof of a durable edge. The profile sets risk_aversion=0.33 — return-tilted but still penalizing variance; whether an even lower λ is genuinely better is a question for forward testing on out-of-sample quarters, not this in-sample curve.
