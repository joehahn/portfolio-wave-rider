Shorter lookbacks (≤ 1 year) chase recent momentum: the μ estimate weighs the most recent surges heavily, so newly accelerating waves get more weight, but the resulting weights are noisier. Longer lookbacks smooth out short-term swings at the cost of diluting recent dynamics. SPY benchmark return over the same window: +27.22%.

Ticker-universe note: a ticker is included only if it has at least LB years of price history at the start of the backtest. NUKZ launched 2024-01-24, so it drops for LB ≥ 2y; RKLB started 2020-11-24, so it also drops at LB = 5y. Lines with different ticker counts aren't strictly comparable — the LB ≥ 2y runs are missing the nuclear-energy bucket entirely, which is the strongest wave tilt over this 12-month window.
